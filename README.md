# db_parkir_uas

##  Deskripsi
**db_parkir_uas** adalah aplikasi sistem database parkir sederhana yang dibuat untuk memenuhi kebutuhan **Ujian Akhir Semester (UAS)**. Aplikasi ini digunakan untuk mengelola data kendaraan parkir secara terstruktur menggunakan database, sehingga proses pencatatan kendaraan masuk dan keluar menjadi lebih rapi dan efisien.

Sistem ini membantu pengguna dalam melakukan pengolahan data parkir, mulai dari penyimpanan data kendaraan hingga pengelolaan informasi waktu parkir.

---

## Tujuan Pembuatan
- Menerapkan konsep **database** pada studi kasus sistem parkir  
- Mengimplementasikan bahasa pemrograman **Python**  
- Melatih penggunaan operasi **CRUD (Create, Read, Update, Delete)**  
- Memahami integrasi aplikasi dengan database  

---

## Teknologi yang Digunakan
- Python 3.x  
- Database (MySQL / SQLite)  
- Library pendukung Python
