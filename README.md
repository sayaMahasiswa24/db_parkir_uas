# Sistem Informasi Parkir
UAS Pengolahan Basis Data

Aplikasi **Sistem Informasi Parkir** ini dibuat untuk memenuhi tugas  
**Ujian Akhir Semester (UAS)** mata kuliah **Pengolahan Basis Data**.

Aplikasi dikembangkan menggunakan **Python** dan **MySQL** dengan konsep  
**CRUD (Create, Read, Update, Delete)**.

## Fitur Aplikasi
- **Create** : Menambahkan data kendaraan parkir
- **Read**   : Menampilkan kendaraan yang sedang parkir
- **Update** : Mengubah status kendaraan keluar
- **Delete** : Menghapus data parkir yang telah selesai
